Descrição do Projeto

Este projeto consiste em um dashboard interativo desenvolvido no Power BI, focado na visualização de dados hospitalares e métricas relacionadas a internações e comorbidades.

Tecnologias Utilizadas

  • Power BI Desktop
  • Power Query (para limpeza e transformação de dados)
  • Visualização Interativa
